☁️ Cloud Sync Setup

Connect a free Supabase database so your checkoffs sync between your phone and laptop in real time.

1. Go to supabase.com → New project (free)
2. Go to SQL Editor → paste and run:
create table schedule (id text primary key, checked jsonb, updated_at timestamptz default now());
3. Go to Settings → API → copy your URL + anon key below
